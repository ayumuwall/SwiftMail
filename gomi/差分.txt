### 1. **データ永続化方針**セクション（アーキテクチャ原則の下）

**現在の記述：**
```markdown
<details>
<summary>データ永続化方針</summary>

  - **Core Data** または **SQLite** を使用する。
      - **Core Data**: オブジェクトグラフ管理やUndo/Redo機能が求められる場合に検討。
      - **SQLite**: より軽量で、SQLによる直接的なデータ操作が望ましい場合に検討。
  - **理由**: 外部依存を避け、macOSネイティブの技術スタックに留めることで、パフォーマンスと長期的なメンテナンス性を確保する。
  - **推奨**: SQLiteをデフォルト使用。理由: Core Dataより軽量で、SQLクエリが検索/同期に適する。Core DataはUndo/Redoが必要な場合のみ（本プロジェクトでは不要）。
  - **実装例**: Swift PackageのSQLite.swiftは禁止（外部ライブラリ制約）。FoundationのSQLite3 APIを直接使用。

</details>

**新しい記述：**
```markdown
<details>
<summary>データベース戦略（SQLite直接使用）</summary>

### SQLite直接使用の方針
- **CoreDataは使用しない**（オーバーヘッドが大きい：起動時50-100ms、メモリ10-15MB）
- **SQLite3のC APIを薄くラップした独自実装**
- **外部SQLiteラッパーライブラリ（FMDB、SQLite.swift等）も使用禁止**

### パフォーマンス設定
```swift
// MailDatabase.swift
func optimizeForPerformance() {
    execute("PRAGMA journal_mode = WAL")        // Write-Ahead Logging
    execute("PRAGMA synchronous = NORMAL")      // バランスの良い同期モード
    execute("PRAGMA cache_size = -64000")       // 64MBのキャッシュ
    execute("PRAGMA temp_store = MEMORY")       // 一時データはメモリに
    execute("PRAGMA mmap_size = 134217728")     // 128MBのメモリマップ
}
```

### 期待されるパフォーマンス
- DB初期化: ~10-20ms（CoreData比 -80%）
- メモリ使用: 基本2-5MB + キャッシュ（CoreData比 -70%）
- 1万件検索: 5-10ms（インデックス使用時）

### データベースファイル
- 場所: `~/Library/Application Support/SwiftMail/mail.db`
- バックアップ: 自動で日次バックアップ（最大7世代）

</details>

// MailDatabase.swift - 最小限のラッパー
import Foundation
import SQLite3

final class MailDatabase {
    private let queue = DispatchQueue(label: "com.swiftmail.db", attributes: .concurrent)
    private var db: OpaquePointer?
    
    init(path: String = Self.defaultPath) throws {
        guard sqlite3_open_v2(path, &db, 
                              SQLITE_OPEN_READWRITE | SQLITE_OPEN_CREATE | SQLITE_OPEN_FULLMUTEX, 
                              nil) == SQLITE_OK else {
            throw MailError.databaseError(String(cString: sqlite3_errmsg(db)))
        }
        
        optimizeForPerformance()
        try createSchema()
    }
    
    private static var defaultPath: String {
        let support = FileManager.default.urls(for: .applicationSupportDirectory, 
                                              in: .userDomainMask).first!
        let appDir = support.appendingPathComponent("SwiftMail")
        try? FileManager.default.createDirectory(at: appDir, 
                                                withIntermediateDirectories: true)
        return appDir.appendingPathComponent("mail.db").path
    }
    
    // トランザクション制御
    func performBatch(_ block: () throws -> Void) rethrows {
        execute("BEGIN IMMEDIATE TRANSACTION")
        do {
            try block()
            execute("COMMIT")
        } catch {
            execute("ROLLBACK")
            throw error
        }
    }
    
    // カーソルベースの効率的なストリーミング
    func streamMessages(limit: Int = 100, offset: Int = 0) -> MessageIterator {
        let query = "SELECT * FROM messages ORDER BY date DESC LIMIT ? OFFSET ?"
        var statement: OpaquePointer?
        sqlite3_prepare_v2(db, query, -1, &statement, nil)
        sqlite3_bind_int(statement, 1, Int32(limit))
        sqlite3_bind_int(statement, 2, Int32(offset))
        
        return MessageIterator(statement: statement)
    }
}

// メモリ効率的なイテレータ
struct MessageIterator: Sequence, IteratorProtocol {
    private let statement: OpaquePointer?
    
    mutating func next() -> Message? {
        guard sqlite3_step(statement) == SQLITE_ROW else { 
            sqlite3_finalize(statement)
            return nil 
        }
        return Message(from: statement)
    }
}
